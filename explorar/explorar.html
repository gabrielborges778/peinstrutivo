Explorar ações existentes no sistema

Após acessar o módulo, na tela principal de solicitação de compra, as ações existentes no sistema são: Incluir, Excluir, Cancelar, Encaminhar, Copiar e Reconfigurar. Cada botão tem a sua respectiva funcionalidade, abaixo será elucidado o que significa ao clicar em cada botão de ação.

  • Clicar no botão para incluir uma nova solicitação de compra.
  • Clicar no botão para excluir uma solicitação de compra.
  • Clicar no botão para cancelar uma solicitação de compra.
  • Clicar no botão para encaminhar uma solicitação para precificação e planejamento.
  • Clicar no botão para copiar uma solicitação de compra existente no sistema.
  • Clicar no botão para reconfigurar uma solicitação de compra.
